# ex08：物联网安防报警器模拟（布防/撤防 + 触摸报警锁定）

## 文件

- 程序文件：[ex08.ino](ex08.ino)

## 实验说明

- 参考：基础 IO、触摸引脚（touchRead）、Web 服务器（ESP32 SoftAP + WebServer）
- 硬件引脚：
	- GPIO2：LED 输出
	- GPIO27（TOUCH7）：触摸输入
- 核心：
	- 网页端提供两个按钮：布防（Arm）与撤防（Disarm）
	- 使用全局状态变量记录系统状态：撤防 / 布防 / 报警锁定
	- 布防状态下，一旦检测到触摸触发阈值：进入“报警锁定”状态
	- 报警锁定后 LED 高频闪烁，即使手松开也不会停止，直到网页点击“撤防”才复位

## 预期效果

- 未布防时触摸引脚无反应
- 点击布防后，一旦触摸引脚触发，LED 进入高频闪烁锁定报警
- 只有点击撤防，LED 才会熄灭并回到撤防状态
