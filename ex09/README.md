# ex09：实时传感器 Web 仪表盘（触摸数值上报 + AJAX 刷新）

## 文件

- 程序文件：[ex09.ino](ex09.ino)

## 实验说明

- 要点：不仅做控制（下发），也做数据采集（上报）
- 硬件引脚：GPIO27（TOUCH7）
- 核心：
	- ESP32 Web Server 下发仪表盘网页（HTML/CSS/JS）
	- 提供数据接口（如 `GET /data`），返回触摸传感器的实时原始数值（纯文本数字）
	- 网页端用 AJAX（定时 `XMLHttpRequest`）不断拉取数值并显示在页面中央

## 预期效果

- 网页中央显示实时跳动的数字
- 手逐渐靠近触摸引脚时数字实时变小；手指离开时数值恢复
- 实现类似仪器仪表的实时监控面板
