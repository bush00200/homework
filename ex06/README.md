## 实验说明

- 硬件引脚：GPIO27 与 GPIO26 各接一个 LED，分别作为两个 PWM 输出
- 核心：初始化两个独立 PWM 输出，并让两个灯占空比呈反相：
	- 灯A：`duty = 0 -> 255`
	- 灯B：`duty = 255 - duty`
- 预期效果：两个 LED 平滑交替渐变，一个变亮同时另一个逐渐变暗，过渡柔和
